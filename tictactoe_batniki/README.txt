Файлы:
run_tests.bat      - запуск тестов на Windows
run_tests_macos.sh - запуск тестов на macOS/Linux
test01.txt ... test30.txt - входные файлы
TESTS_LIST.txt     - список тестов с содержимым

Windows:
1. Перейдите в каталог tictactoe_batniki
2. Положите tic_tac_toe.exe на файловый уровень ниже
3. Запустите run_tests.bat.
4. Смотрите results.txt.

macOS/Linux:
1. Перейдите в каталог tictactoe_batniki
2. Положите исполняемый файл tic_tac_toe на файловый уровень ниже.
3. Выполните:
   chmod +x ./run_tests.sh
   ./run_tests.sh
4. Смотрите results.txt.
