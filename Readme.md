# TicTacToe Strategy Solver

Программа вычисляет оптимальную стратегию для выбранного игрока и сохраняет дерево оптимальных продолжений в формате DOT.

## Сборка

Из корня проекта `tic-tac-toe`:

```bash
cmake -S . -B build
cmake --build build
```

После сборки исполняемый файл появляется по пути `./tic_tac_toe`.

## Запуск

```bash
./tic_tac_toe input.txt output.dot
```

Первый аргумент: путь к входному текстовому файлу.

Второй аргумент: путь к выходному файлу в формате DOT.

## Формат входных данных

Входной файл должен содержать ровно 5 строк:

1. первая строка поля;
2. вторая строка поля;
3. третья строка поля;
4. игрок, для которого строится стратегия: `X` или `O`;
5. игрок следующего хода: `X` или `O`.

Правила для первых трех строк:

- каждая строка должна содержать ровно 3 символа;
- допустимые символы: `X`, `O`, `.`;
- `.` обозначает пустую клетку.

Пример входного файла:

```text
O..
.O.
..X
O
X
```

## Формат выходных данных

Программа создает файл `output.dot` с деревом всех оптимальных продолжений из
переданной позиции.

- при успешном выполнении программа завершает работу с кодом `0`;
- при ошибке программа печатает сообщение в стандартный вывод и завершает
  работу с ненулевым кодом.
